# LanRen-CLI
Lan(懶)Ren(人) CLI

# 安裝 commander
```
npm install commander --save
```

# 安裝 LanRen-CLI
```
npm link
```
這個指令將會幫助你把這個資料夾放進你的全域node module 中，這樣妳就不用部屬到 npm 上才能使用。

# 解安裝 
```
npm unlink LanRen-CLI


npm rm --global LanRen-CLI

npm ls --global LanRen-CLI
```

# 查看版本號
```
lr -V
or
lr --version
```

# 查看說明
```
lr -h
or
lr --help
```
